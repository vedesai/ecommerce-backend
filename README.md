# Crest & Thread E-commerce Backend

Spring Boot backend API for the Crest & Thread e-commerce platform.

## Tech Stack
- Java 21
- Spring Boot 3.3.0
- Spring Data JPA
- H2 (dev) / PostgreSQL (prod)
- MapStruct
- Swagger/OpenAPI

## Getting Started

```bash
./mvnw spring-boot:run
```

API will be available at `http://localhost:8080/api`

## API Documentation
- Swagger UI: http://localhost:8080/api/swagger-ui.html
- OpenAPI docs: http://localhost:8080/api/api-docs
